Claude Code, Codex, Antigravity — Mac & Windows 설치 방법

사전 준비

Node.js 22+ 설치

Claude Code와 Codex는 Node.js 22 이상이 필요합니다.

Terminal
brew install node@22

또는 nodejs.org에서 .pkg 다운로드

PowerShell (관리자)
winget install OpenJS.NodeJS.LTS

또는 nodejs.org에서 .msi 다운로드

터미널 열기

설치 명령어를 입력할 터미널을 엽니다.

1Cmd + Space 로 Spotlight 열기
2"터미널" 입력 후 Enter
1시작 버튼 우클릭
2"Windows PowerShell (관리자)" 선택

Claude Code

Anthropic
1Node.js 22+ 설치 확인
2터미널에서 설치:
Terminal
npm install -g @anthropic-ai/claude-code
3실행:
Terminal
claude
4첫 실행 시 브라우저가 열리며 Anthropic 계정 로그인
