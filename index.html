Claude Code, Codex, Antigravity — Mac & Windows 설치 방법

사전 준비

Node.js 22+ 설치

Claude Code와 Codex는 Node.js 22 이상이 필요합니다.

Terminal
brew install node@22

또는 nodejs.org에서 .pkg 다운로드

PowerShell (관리자)
winget install OpenJS.NodeJS.LTS

또는 nodejs.org에서 .msi 다운로드

터미널 열기

설치 명령어를 입력할 터미널을 엽니다.

1Cmd + Space 로 Spotlight 열기
2"터미널" 입력 후 Enter
1시작 버튼 우클릭
2"Windows PowerShell (관리자)" 선택

Claude Code

Anthropic
1Node.js 22+ 설치 확인
2터미널에서 설치:
Terminal
npm install -g @anthropic-ai/claude-code
3실행:
Terminal
claude
4첫 실행 시 브라우저가 열리며 Anthropic 계정 로그인

Antigravity

Google

Google이 만든 AI 에이전트 IDE (VS Code 기반). 프리뷰 기간 중 무료. Gemini 3 Pro 모델 기본 탑재.

1공식 사이트에서 다운로드:

antigravity.google 접속 → Mac용 (.dmg) 다운로드 → 설치

2실행:

Applications에서 Antigravity 실행 → Google 계정 로그인

3(선택) CLI에서도 실행 가능:
Terminal
antigravity

Command Palette (Cmd+Shift+P) → "Shell Command: Install" 먼저 실행

1공식 사이트에서 다운로드:

antigravity.google 접속 → Windows용 (.exe) 다운로드 → 설치

2실행:

시작 메뉴에서 Antigravity 검색 → 실행 → Google 계정 로그인

3(선택) CLI에서도 실행 가능:
PowerShell
antigravity

설치 시 PATH에 자동 추가됨

필요 구독 무료 (프리뷰), Google 계정만 필요
공식 사이트 antigravity.google
